## Cómo interpretar la prioridad
- P1 STOCKOUT → compra inmediata, sin stock disponible
- P2 CRITICO  → incluir en próxima orden de compra
- P3 BAJO     → planificar para próximo trimestre
- P4 OK       → sin acción necesaria
